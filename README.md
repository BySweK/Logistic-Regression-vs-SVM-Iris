🌸 Iris Species Classification Analysis
This project explores the classic Iris Dataset using machine learning techniques. I have implemented and compared two different classification models to predict iris species based on their physical features.

📊 Project Overview
The workflow for this analysis includes:

Data Exploration: Analyzing relationships between petal and sepal features using Pandas and Seaborn.  

Visualization: Utilizing scatterplots to visualize how different species cluster based on their dimensions.  

Modeling:

Logistic Regression: A linear model used for classification.  

Support Vector Machines (SVM): Applied to find the optimal hyperplane for species separation.  

Performance Evaluation: Measuring model accuracy to determine the most effective approach.  

🛠️ Tech Stack
Python 3

Pandas & Numpy: For data manipulation and numerical operations.  

Matplotlib & Seaborn: For data visualization.  

Scikit-learn: For implementing machine learning algorithms and preprocessing.  

📁 Dataset Information
The dataset contains 150 samples from three species of Iris (Setosa, Versicolor, and Virginica). Four features were measured from each sample: the length and the width of the sepals and petals.
